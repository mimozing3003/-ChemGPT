## 🌟 Overview

**ChemGPT** is an interactive, futuristic AI Chemistry Copilot built for students, researchers, and scientists. It bridges the gap between static chemical textbooks and interactive, 3D molecular intelligence.

### Key Features

1. **AI Chemistry Workspace (`chat.html`)**
   - Real-time AI chemistry chat interface
   - Interactive 3D molecular structures canvas
   - Command palette (`Ctrl+K` / `Cmd+K`)

2. **ChemVerse Encyclopedia (`encyclopedia.html`)**
   - **Global Chemistry Search**: Search 118 elements, famous molecules, formulas, and properties with instant autocomplete.
   - **Interactive Periodic Table**: Filter by element category (*Alkali Metals, Transition Metals, Noble Gases, etc.*).
   - **3D Element Dashboard Panel**: Click any element to view its 3D atom animation (Three.js), electron configuration, physical properties, and compounds.
   - **Side-by-Side Element Comparison Studio**: Compare density, atomic mass, electronegativity, melting points, and valence electrons.
   - **Chemistry Calculator Suite**: Interactive modals for Molar Mass, pH & Concentration, Dilution ($C_1V_1 = C_2V_2$), and Ideal Gas Law ($PV=nRT$).

3. **Modern Brutalist & Glassmorphic UI**
   - Designed with deep space dark mode (`#080810`), neon emerald highlights (`#4edea3`), and warm amber accents (`#ffb95f`).
   - Smooth GSAP scroll reveals & Three.js particle backgrounds.
